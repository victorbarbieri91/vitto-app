# API Reference

Esta pasta conterá a documentação automática da API gerada pelo TypeDoc.

## Como Gerar

Para gerar a documentação da API:

```bash
npm run docs:api
```

A documentação será gerada nesta pasta automaticamente a partir do código TypeScript em `src/`.

## O que está incluído

- Classes e interfaces
- Funções e métodos
- Types e enums
- JSDoc comments

---

> **Nota**: Esta documentação é gerada automaticamente. Não edite manualmente os arquivos gerados.
